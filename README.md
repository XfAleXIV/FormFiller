# FormFiller - Instructions
## Objetivo
* Crear una herramienta para facilitar el rellenar documentos.

## Emphasis

Esta Herramienta es solo una **BETA**. Posee problemas, como el guardado incorrecto cuando se trabaja en la nube.


## ¿Cómo usarlo?

### Ejemplo

Como ejemplo usaremos el documento subido como "Gestion-de-Cobro.dotx - Ejemplo de Relleno". Esta plantilla tiene las etiquetas que reemplazaremos. 

**1.** **Elige una plantilla** con el botón "Seleccionar Plantilla"

**2.** **Rellena con los datos deseados**

Los datos en la primera tabla son el Encabezado, son los datos generales que compartiran todos los documentos.

La segunda tabla son todos los datos especificos de cada documento, cada fila representa un documento.

**3.** Presiona el botón **Hacer Copia(s)** 
    1. Item 3a
    2. Item 3b

### Usa tu propia plantilla

Si quieres usar esta herramienta con otras plantillas deberas escribir tus etiquetas en la plantilla y en la herramienta. Para hacerlo presiona el botón "_Config_". Aparecerán Una columna y fila amarilla con las etiquetas, agrega o reemplaza a tu gusto. Si necesitas más espacio, ve a la ultima fila e inserta otra. y Repetir los pasos de ls sección anterior.

## Consideraciones

Al descargar la herramienta necesitarás activar los macros de Excel. Más información en este enlace [**Habilitar o deshabilitar macros en archivos de Microsoft 365**](https://support.microsoft.com/es-es/office/habilitar-o-deshabilitar-macros-en-archivos-de-microsoft-365-12b036fd-d140-4e74-b45e-16fed1a7e5c6)
